# Neon Snake - CodeByArt Edition

**A premium, high-performance HTML5 Arcade Game optimized for instant embedding.**

Developed by **CodeByArt**, this project re-imagines the classic Snake game with a futuristic Cyberpunk aesthetic, robust responsive design, and a zero-dependency architecture suitable for WordPress, GitHub Pages, or any web platform.

![CodeByArt Logo](https://codebyart.com/wp-content/uploads/2025/10/CodeByArt-Logo.png)

## ‚ö° Key Features

*   **Single File Architecture:** The entire game (Logic, UI, Styles, Audio) is bundled into one `index.html` file. No build steps, node_modules, or complex deployments required.
*   **Adaptive "Game Zone" Layout:**
    *   **Desktop:** Immersive "Arcade Cabinet" experience with CRT scanlines, glassmorphism UI, and 4K-ready rendering.
    *   **Mobile:** Ergonomic "Handheld Console" layout with large, haptic-feedback touch controls and bottom-aligned inputs.
*   **Synthesized Audio Engine:** Custom-written audio controller that generates sound effects in real-time using the Web Audio API. No heavy `.mp3` or `.wav` assets to load.
*   **Player Customization:**
    *   4 Difficulty Levels (Easy, Medium, Hard, Insane).
    *   4 Neon Themes (Green, Cyber Blue, Plasma Pink, Gold Glitch).
*   **Persistent Data:** LocalStorage integration saves high scores automatically.

### Option 1: WordPress (Copy & Paste)
1.  Open `index.html` in any text editor.
2.  Copy the entire code.
3.  Go to your WordPress Page/Post.
4.  Add a **"Custom HTML"** block.
5.  Paste the code and Publish.

### Option 2: GitHub Pages / Static Hosting
1.  Upload `index.html` to your repository.
2.  Enable GitHub Pages pointing to the file.
3.  The game is instantly live.
